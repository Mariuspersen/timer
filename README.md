# timer
Just a super simple timer

## Instructions

To compile:
```
make
```

Usage:
```
usage:  timer <operation> [...]
operations:
        timer -h <hours>
        timer -m <minutes>
        timer -s <seconds>

 use timer for available options
```

To install simply type:
```
make && sudo make install
```
To uninstall you can type:
```
sudo make uninstall
```
